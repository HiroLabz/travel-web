# **App Name**: WanderNest

## Core Features:

- User Authentication: Secure user login and registration with email/password and Google authentication.
- Household Management: Create and manage households, including adding and managing members with different roles.
- Trip Planning Wizard: Guide users through the process of planning a trip, including destination selection and setting the overall vibe.
- AI-Powered Itinerary Generation: Use Google Gemini AI as a tool to generate trip itineraries based on destination, duration, and vibe input by the user.
- Interactive Trip Itinerary: Display the generated or manually created itinerary in an interactive and easy-to-navigate format using a rich text editor.
- Document Vault: Upload and manage essential travel documents like passports and tickets, associating them with specific trips.
- Real-time Data Sync: Use Firestore to ensure that all trip and household data is synchronized in real-time across all devices.

## Style Guidelines:

- Primary color: Deep turquoise (#46A3FF), a calm and trustworthy blue that evokes travel.
- Background color: Very light blue (#F0F8FF), providing a clean, airy feel.
- Accent color: Vibrant indigo (#663399) for interactive elements, providing contrast.
- Body text: 'PT Sans', a humanist sans-serif offering a modern and welcoming aesthetic; this font also works as a header, allowing for single-font MVP simplicity.
- Note: currently only Google Fonts are supported.
- Use consistent and clear icons from Lucide React for UI elements related to travel planning, documents, and household management.
- Employ a clean and intuitive layout using Tailwind CSS, focusing on readability and ease of navigation.
- Incorporate subtle animations for user interactions, such as loading screens and transitions, to enhance the user experience.
